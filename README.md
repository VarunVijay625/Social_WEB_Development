# Social WEB
## Team: Da Web Inferno
How to run this program: Open choice.html in your preferred browser. Navbar tabs at the top of the webpage allow you to explore current capabilities.

If user not yet in the database, they now have the possibility to create an account,
when navigating through the choice.html page:
- first step is to click on create an account
- When redirected to the account page the user chooses "my name is not listed" from the dropdown button that lists all the people in the web so far
- The user also filled out the rest of the form: name + initial, username and password
- After they hit submit and all of their info is directly entered to our database

Current capabilities:  
+ User can create an account.  
+ User can see web based on current database.  
+ User can query for certain relationships to certain people.
